API Reference

Auth routes

POST/api/Registro/user signup
POST/api/Login/user signin
POST/api/Logout/user signout

Users routes

All access

GET/api/Home/retrieve all available products

Requires login as user

GET/api/Cart/retrieve all products in the user's shopping cart
POST/api/Cart/add the specified product to the user's shopping cart
GET/api/Purchase/retrieve all products purchased by the user
POST/api/Purchase/move all the products from the user's shopping cart to the user's purchased products

Requires login as admin

GET/api/Users/retrieve all users
GET/api/Roles/retrieve all existing roles
POST/api/Producto/add a new product to the store
